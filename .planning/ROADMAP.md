# Roadmap: Autopitch

## Overview

Autopitch transforms a single Excel workbook into a boardroom-ready consulting deck. The build follows the pipeline's hard dependency chain: validated financial data must exist before metrics can be computed, metrics must exist before charts and narrative can be generated, and both must exist before the deck can be assembled and exposed through interfaces.
