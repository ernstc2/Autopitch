# Roadmap: Autopitch

## Overview

Autopitch transforms a single Excel workbook into a boardroom-ready consulting deck. The pipeline parses financial data, computes metrics, generates AI narrative, and assembles a polished 11-slide PPTX. Deployed as a live portfolio demo on Streamlit Cloud.
